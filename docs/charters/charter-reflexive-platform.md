# Charter — Reflexive / Platform (the outermost strange loops)

*The cluster where antigen becomes infrastructure the stack is built ON, and where it immunizes its own
ecosystem — the strange loops close. The wire-format precondition is already shipped.*

**Tier:** deep-future (chartered expedition). The wire-format (Finding schema) is **build-now**.

---

## Organ-identity (what this IS)

The immune layer the stack is built on, and the immune system of the immune system. Two reflexive loops:
**platform** — antigen as an LSP-of-failure-class-memory that other tools (and AI agents *at generation
time*) query, which is the asymmetry's cure at the source (the agent asks "is what I'm about to generate a
known failure-class?" before writing it); and **self-evolution** — who immunizes antigen's own ecosystem
(Jerne's idiotypic network: the immune system regulating itself), the loop closing on itself.

## The dreams in this cluster

- **`dream/antigen-as-platform-immune-infrastructure`** — the LSP-of-failure-class-memory; the
  consume-from-outside API + wire-format. **The wire-format precondition is ALREADY SHIPPED: the Finding
  schema (ADR-039 §C) with `schema_version` + the additive-optional forward-compat rule** (added precisely
  so this external platform contract is versionable — adversarial finding #12). **Keep the core callable
  now** — the scanner + the Finding stream are the platform's API surface; treat them as the contract.
- **`dream/evolve-immune-system-of-the-immune-system`** — `#[evolve]`; who immunizes antigen's own
  ecosystem (Jerne's idiotypic network). The strange loop: antigen catching antigen's own failure-classes
  (the dogfood spine is the seed; this is its full reflexive form).

## Dependencies (what unblocks this)

- **The Finding schema as a versioned wire-format (SHIPPED, ADR-039 §C)** — the platform's whole premise is
  external consumers subscribing to a stable contract; `schema_version` + the additive-optional rule make
  it stable. This precondition is *already met* by the build-now seam (that's why the seam was load-bearing).
- **The learning core** — `#[evolve]` (antigen maturing fingerprints for its own ecosystem) is the
  maturation engine pointed reflexively; it needs the engine first.

## Could-combine-with

- The **prevention-transmission charter** — antigen-as-platform (external tools query it) and
  antigen-as-felt-sense (the dev's interior surface) are the same wire-format consumed by different clients
  (a CI bot vs an editor).
- The **runtime-afferent charter** — an agent querying antigen at generation time is the *efferent* arm
  (matured knowledge exported to the point of generation), the dual of the afferent drainage.

## Buildability / effort scoping

- **The platform API: the SUBSTRATE is build-now** (the Finding schema is the wire-format; the scanner is
  callable). Exposing it as a stable consume-from-outside surface (an LSP server, a queryable index) is
  MODERATE — a real service wrapping existing machinery, but no new core.
- **Agent-queries-antigen-at-generation: the highest-leverage slice** — an MCP/LSP endpoint an AI agent
  hits before generating ("is this a known failure-class?") is the asymmetry's actual cure at the source.
  Buildable on the shipped scanner + schema; MODERATE.
- **`#[evolve]` / idiotypic self-immunization: HARD + reflexive** — needs the learning core + a notion of
  antigen's own ecosystem as a corpus. The deepest-future item; charter last.

## Invitation to deepen

Jerne's idiotypic network: the immune system doesn't just recognize pathogens, it recognizes *itself*
recognizing — antibodies against antibodies, a self-regulating web. The antigen cognate is the strangest
loop on the whole map: antigen's fingerprints are themselves code, so antigen can scan its own fingerprints
for failure-classes, mature a defense against its own bugs, and forward-deploy it — the immune system of
the immune system, all the way down. The dogfood spine (antigen catching its own `AuditVerdictComputedButNotDelivered`,
its own gem/forgery bugs) is the seed; this charter is where the loop becomes the system's defining shape.
