# Charter — Prevention / Transmission (immunize before the harm; antigen as a felt sense)

*The cluster that moves antigen upstream of the failure — the marked codebase as a curriculum, and antigen
as a proprioceptive sense the dev HAS rather than a tool they run.*

**Tier:** deep-future (chartered expedition). One hint — felt-WEIGHT — is **design-now**.

---

## Organ-identity (what this IS)

Vaccination + proprioception. Two faces: **transmission** — the densely-marked masterclass codebase IS a
curriculum of how this kind of code fails, so a new contributor (or a compacted agent — "compaction = a new
hire every session") is immunized by reading the marks; and **the felt sense** — antigen not as a `cargo`
command you run but as an ambient sense the dev *has*, proprioceptive, at keystroke speed. The only mode
that beats the generation-inspection asymmetry at the speed the asymmetry happens.

## The dreams in this cluster

- **`dream/teaching-instrument-onboarding-curriculum`** + **`dream/immune-memory-as-onboarding`**
  (converged) — the marked codebase IS a curriculum. **Near-slice: a curriculum auto-generated from
  antigen's own dogfood marks** (the masterclass teaches itself). "Compaction = a new hire every session"
  is the agentic framing — every fresh agent reads the marks and is onboarded by them.
- **`dream/felt-immune-system-ambient-sense`** — antigen as a SENSE the dev HAS (ambient, proprioceptive),
  not a tool they run. **DESIGN-NOW HINT: store dread/aura magnitude as a continuous felt-WEIGHT, not just a
  category** — the Finding schema's `severity`/`magnitude` (now universal, ADR-039 §C) is the substrate;
  leave it continuous-capable so a future ambient surface can render a felt-gradient.

## Dependencies (what unblocks this)

- **The masterclass dogfood (build-now, beta.2)** — the curriculum is generated FROM the masterclass's own
  marks, so the teaching-instrument is downstream of `dogfood/comprehensive-front-line-coverage` +
  `specimen/pathology-garden`. The curriculum-from-marks slice becomes buildable as soon as the masterclass
  ships.
- **The felt-WEIGHT design-now hint** — already satisfied by the universal `severity` field (keep it
  continuous-capable, don't collapse to a 3-value enum at the schema if a future ambient surface wants a
  gradient). No new substrate; a forward-compat note.

## Could-combine-with

- The **reflexive-platform charter** — antigen-as-a-felt-sense is antigen-as-platform from the dev's
  interior view (an IDE/editor surface that renders the ambient gradient). Same wire-format (the Finding
  schema), different consumer.

## Buildability / effort scoping

- **Curriculum-from-marks: MODEST** — antigen already has the marks (the dogfood antigens + the masterclass
  coverage); generating a curriculum is a structured read-and-render over existing structured data. Real
  but bounded; becomes buildable when the masterclass ships.
- **Felt-sense ambient surface: HARD (an editor/IDE integration)** — a continuous proprioceptive surface is
  an LSP/editor component, the deepest version. The design-now part (keep severity continuous-capable) is
  free; the surface itself is a real integration.

## Invitation to deepen

The asymmetry antigen exists to fix (generation is easy, inspection is hard) happens at keystroke speed —
so the *only* mode that truly fixes it is one that operates at keystroke speed too: not a tool the dev
remembers to run, but a sense the dev cannot help but have, like proprioception. You don't "run" your sense
of where your hand is; you just know. The deepest antigen is the one the dev stops noticing because it has
become part of how they perceive their own code.
