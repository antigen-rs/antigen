# Charter — Beyond Code (research-undefined: any generated artifact)

*The cluster where antigen's immune system extends past Rust source to ANY generated artifact with an exact
substrate. The biggest identity question on the map — RULED in-identity (Tekgy), but scope-undefined: it
needs a study before a build.*

**Tier:** research-undefined. **Identity: RULED YES, charter-not-beta.2 (Tekgy). Scope: a study, not a build.**

---

## Organ-identity (what this IS)

The immune system generalized past code. The thesis — failure-class memory made structural and co-native —
is not intrinsically about Rust; it's about *any* artifact a generator produces faster than an inspector
can check. Papers, contracts, datasets, AI-training-data: each has its own failure-classes that a generator
re-derives and an inspector struggles to catch — the same generation-inspection asymmetry, one domain over.

## The dreams in this cluster

- **`dream/beyond-code-any-generated-artifact`** — the immune system for any generated artifact. **The
  identity-fork (bigger than runtime; Tekgy ruled YES, charter-not-beta.2).**
- **`dream/document-immune-system-sidecar-markdown`** — the first concrete domain: structured docs, via a
  sidecar-to-markdown with synced line-refs. **ADR-in-markdown first** (antigen's own decisions.md is the
  test case — the doc that drifts from its impl IS a substrate-alignment failure antigen already names).
- **`dream/nearest-second-organism-data-and-research-code`** — the nearest non-Rust domain with an exact
  substrate (data pipelines / research code), the natural second organism.

## The honest gate (why this is research-undefined, not deep-future)

**Only where the domain has an EXACT substrate** — else it's RAG, which the vision explicitly rejects.
Antigen's whole power is that a fingerprint matches an *exact structural tell*, not a fuzzy semantic
similarity. A markdown sidecar with synced line-refs HAS an exact substrate (the line is the structural
anchor); a free-form "is this contract risky?" does NOT (it's semantic judgment = RAG). So the charter's
first job is not to build — it is to **STUDY which beyond-code domains have an exact substrate** and which
are RAG-in-disguise. That study is the research-undefined work; the build follows only for the domains that
pass the exact-substrate gate.

## Dependencies (what unblocks this)

- **A harbor-master scope ruling** (the identity is settled YES; the *scope* — which domains, in what
  order — is the open study). This is the one cluster gated on a study rather than a prior build.
- **The structured-first / prose-marks-the-gap discipline** (the exactness-gate made practical): a domain
  qualifies if its artifact is structured-first (the line-ref, the schema, the typed field) with prose only
  marking the gap — not prose-first with structure bolted on.

## Buildability / effort scoping

- **The STUDY is the deliverable first** — scope which domains have an exact substrate. Effort: a research
  pass, not a build. (Cheap to start, defines everything after.)
- **ADR-in-markdown (the first qualifying domain): MODEST** — antigen already names doc-vs-impl drift
  (`RatifiedSpecDriftFromImpl`, a shipped dogfood class); the sidecar-with-synced-line-refs is a real but
  bounded extension. This is the one slice with a clear exact substrate today.
- **General beyond-code: UNDEFINED** — each domain is its own unopened immune system; effort is
  per-domain and gated on the exact-substrate study.

## Invitation to deepen

The dreamer's frontier note: "every domain in beyond-code is its own unopened immune system. There is no
edge. There's only the next chart." The deepest question this charter holds: is the immune-system shape
(sense → compare → route → act → feedback, sized by requisite variety) *universal* to any
generation-inspection-asymmetric domain, or is it Rust-specific dressed in biology? If universal, antigen's
real identity is not "a Rust tool" but "the structural memory of failure-classes for any co-native
substrate" — and that is the biggest version of the dream, the one the whole voyage kept making bigger.
